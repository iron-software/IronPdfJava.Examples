### Manipulating PDF Documents

Using IronPDF, you have seamless control over PDF documents. This includes adding new pages, copying pages individually or in ranges, and removing unwanted pages effortlessly.

#### Add Pages

```java
// Import essential packages
import java.io.IOException;
import java.nio.file.Paths;
import com.ironsoftware.ironpdf.PdfDocument;

// Load existing PDF and create a new PDF from HTML for the cover page.
PdfDocument mainPdf = PdfDocument.fromFile(Paths.get("assets/sample.pdf"));
PdfDocument coverPage = PdfDocument.renderHtmlAsPdf("<h1>Cover Page</h1><hr>");

// Insert the cover page at the beginning of the main PDF.
mainPdf.prependPdf(coverPage);
mainPdf.saveAs(Paths.get("report_with_cover.pdf"));
```

#### Copy Pages

```java
// Copy the first page of a PDF and save as a new document.
import java.io.IOException;
import java.nio.file.Paths;
import com.ironsoftware.ironpdf.PdfDocument;

```

#### Delete Pages


### Custom Dimensions for PDF Documents

IronPDF provides the capability for developers to tailor PDF dimensions to their specific requirements, extending beyond the typical A4 paper size (8½ by 11 inches or 21.59 by 27.94 cm).

```java
import com.ironsoftware.ironpdf.*;  
import com.ironsoftware.ironpdf.render.*;  
import java.io.IOException;  
import java.nio.file.Paths;

// Define HTML content
String htmlContent = "<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>";  

// Initialize ChromePdfRenderOptions object to customize PDF rendering
ChromePdfRenderOptions options = new ChromePdfRenderOptions();  
// Set custom paper size rather than default sizes
options.setPaperSize(PaperSize.Custom);  

/*
 * You can specify the paper size in various units:
 * 1. setCustomPaperWidth(width) / setCustomPaperHeight(height): Specify in inches
 * 2. setCustomPaperSizeInCentimeters(width, height): Use centimeters
 * 3. setCustomPaperSizeInMillimeters(width, height): Use millimeters
 * 4. setCustomPaperSizeInPixelsOrPoints(width, height): Use pixels or points
 */
options.setCustomPaperSizeInCentimeters(13.97, 13.97);  

// Render HTML as PDF with customized dimensions and save it
PdfDocument.renderHtmlAsPdf(htmlContent, options).saveAs(Paths.get("assets/CustomPaperSize.pdf"));
```

### Adjusting PDF Page Orientation

IronPDF for Java offers functionality to alter the orientation of pages within both newly created and existing PDF documents. Typically, PDFs generated using IronPDF default to a portrait layout. However, developers have the flexibility to switch this setting to landscape by employing the `ChromePdfRenderOptions` when transforming various content types like HTML, RTFs, or URLs into PDF format. For more information, visit the detailed guide on [setting PDF page orientation](https://ironpdf.com/java/examples/pdf-page-orientation/).

```java
import com.ironsoftware.ironpdf.PdfDocument;
import com.ironsoftware.ironpdf.edit.PageSelection;
import com.ironsoftware.ironpdf.page.PageRotation;
import com.ironsoftware.ironpdf.render.*;
import java.io.IOException;
import java.nio.file.Paths;

// Setting the orientation for newly created PDF documents to either portrait or landscape.
ChromePdfRenderOptions options = new ChromePdfRenderOptions();
options.setPaperOrientation(PaperOrientation.LANDSCAPE);
PdfDocument newDocument = PdfDocument.renderUrlAsPdf("https://ironpdf.com", options);
newDocument.saveAs(Paths.get("assets/LandscapePdf.pdf"));

// Adjusting the orientation of existing PDF documents.
PdfDocument existingDocument = PdfDocument.fromFile(Paths.get("assets/example.pdf"));

// Checking the orientation of the first page in the existing PDF.
PageRotation initialRotation = existingDocument.getPagesInfo().get(0).getPageRotation();
System.out.println(initialRotation);

// Rotating the first page of the existing PDF by 90 degrees clockwise.
existingDocument.rotatePage(PageRotation.CLOCKWISE_90, PageSelection.firstPage());

// Rotating all pages in the existing PDF by 270 degrees clockwise.
existingDocument.rotateAllPages(PageRotation.CLOCKWISE_270);

existingDocument.saveAs(Paths.get("assets/ExistingPdfRotated.pdf"));
```

For additional details, please explore the IronPDF's online guide on [setting PDF orientation](https://ironpdf.com/java/examples/pdf-page-orientation/).

## Transforming PDF Documents into Images

IronPDF possesses the ability to transform any PDF document—whether freshly generated, sourced from external content, or tailored with customized headers, footers, and margins—into images. These images can then be efficiently stored on a file system, archived in a database, or shared across network systems.

Here is the paraphrased section with the relative URL paths resolved to ironpdf.com:

```java
import com.ironsoftware.ironpdf.PdfDocument;
import com.ironsoftware.ironpdf.edit.PageSelection;
import com.ironsoftware.ironpdf.image.ToImageOptions;
import javax.imageio.ImageIO;
import java.awt.image.BufferedImage;
import java.io.File;
import java.io.IOException;
import java.nio.file.Paths;
import java.util.List;

// Load a PDF file to be processed.
PdfDocument pdf = PdfDocument.fromFile(Paths.get("assets/composite.pdf"));

// Retrieve images from all pages in the PDF document.
List<BufferedImage> extractedImages = pdf.toBufferedImages();

// Configure settings to adjust the maximum dimensions and resolution (DPI) of the images.
ToImageOptions options = new ToImageOptions();
options.setImageMaxHeight(100);
options.setImageMaxWidth(100);

// Extract images from the PDF, applying the configured dimensions and selection criteria.
// PageSelection options include: allPages, lastPage, firstPage, singlePage(int pageIndex),
// and pageRange(int startingPage, int endingPage)
List<BufferedImage> resizedImages = pdf.toBufferedImages(options, PageSelection.allPages());

// Store each extracted and resized image as a separate PNG file.
int imageCount = 1;
for (BufferedImage image : resizedImages) {
    String imagePath = "assets/images/" + imageCount++ + ".png";
    ImageIO.write(image, "PNG", new File(imagePath));
}
```

## Images and Text Extraction

IronPDF offers robust tools for the extraction of both images and text from PDF files, allowing developers to access and manipulate content effectively. Learn more about these capabilities in the [content extraction documentation](https://ironpdf.com/java/examples/extract-text-from-pdf/).

The method `extractAllText`, accessible through any instance of `PdfDocument`, retrieves all textual content from a PDF and presents it as a single string. For images, the method `extractAllImages` compiles a list of all images embedded within the PDF, with each image represented as a `BufferedImage`. To obtain the images in their raw byte format, the method `extractAllRawImages` can be utilized.

### Extracting Text and Images

IronPDF provides robust capabilities to [extract textual content](https://ironpdf.com/java/examples/extract-text-from-pdf/) from PDF documents, which can be used to pull text data for processing or analysis.

The `extractAllText` function on all `PdfDocument` objects retrieves a string containing all text within the PDF, making it ideal for data extraction or indexing documents for search capabilities.

```java
PdfDocument pdf = PdfDocument.renderUrlAsPdf("https://unsplash.com/");
String allText = pdf.extractAllText();
System.out.println("Extracted text from the PDF: " + allText);
```

For image extraction, IronPDF enables developers to [retrieve all images](https://ironpdf.com/java/examples/extract-text-from-pdf/) from a PDF document, each represented as a `BufferedImage` object. Additionally, raw image bytes can be obtained using the `extractAllRawImages` method, allowing for further manipulation or storage.

```java
PdfDocument pdf = PdfDocument.renderUrlAsPdf("https://unsplash.com/");
try {
    List<BufferedImage> extractedImages = pdf.extractAllImages();
``` 

## Managing Forms in PDFs

IronPDF for Java simplifies the process of interacting with forms within PDF documents. The `FormManager` class allows developers to effortlessly assign and access values in form text fields. To assign a value, developers can utilize the `setFieldValue` function by specifying the field name and the desired value.

Retrieving values from form fields is equally straightforward. Developers can directly access `FormField` objects within the `FormManager` collection by specifying the appropriate field name or index. This functionality provides significant flexibility, enhancing the ease of working with forms in dynamic and interactive PDF documents.

### Form Creation and Modification

IronPDF Java grants developers the tools to effectively manage form fields within PDF documents. This ease of use is vital for handling both the creation of new forms from HTML content and the alteration of existing forms.

#### Generating New Forms from HTML

Below is the workflow to craft a PDF form directly from HTML markup:

```java
import com.ironsoftware.ironpdf.PdfDocument;  
import com.ironsoftware.ironpdf.render.ChromePdfRenderOptions;  
import java.io.IOException;  
import java.nio.file.*;

// Define the location to save the generated PDF form
Path outputLocation = Paths.get("assets/BasicForm.pdf");  
String formHTML = "<html>"  
  + "<body>"  
  + "<h2>Editable PDF Form</h2>"  
  + "<form>"  
  + "First name: <br> <input type='text' name='firstname' value=''> <br>"  
  + "Last name: <br> <input type='text' name='lastname' value=''>"  
  + "</form>"  
  + "</body>"  
  + "</html>";  

// Set up rendering options to enable form creation from HTML
ChromePdfRenderOptions renderOptions = new ChromePdfRenderOptions();  
renderOptions.setCreatePdfFormsFromHtml(true);  

// Convert HTML to a PDF form and save it
PdfDocument.renderHtmlAsPdf(formHTML, renderOptions).saveAs(outputLocation);  
```

#### Modifying Existing Forms

To modify an existing PDF form, you can change the values of its fields as follows:

```java
import com.ironsoftware.ironpdf.PdfDocument;

// Save the updated PDF form
form.saveAs(Paths.get("assets/BasicForm_Filled.pdf"));
```

## Printing PDF Documents

With IronPDF, incorporating printing capabilities into your applications is straightforward. By employing the `print` method, developers can activate the default printing dialog of the operating system. This dialog permits the adjustment of various print settings, including selecting the printer, specifying paper sizes, and determining the number of copies desired. For further guidance, you can explore how to incorporate PDF printing into your projects on IronPDF's [detailed example page](https://ironpdf.com/java/examples/bookmarks/).

```java
import com.ironsoftware.ironpdf.PdfDocument;
import java.awt.print.PrinterException;

// Generate a PDF document using HTML content
```
